Project : Counter Burger

Description :
The project is aimed at reproducing the online ordering system for the Counter Burger.This project is implemented using Scrum methodology and is divided into two sprints. Following are the backlog items covered in the sprint

• Customer Places Order
• Quote Order for Customer
• Manager Views Inventory
• Manager Views Sales 

Technical Details

UML Modeling Tool: Astah Community
Front end: Microsoft ASP.Net
Middle tier: C#, ADO.Net
Backend: SQL Server 2008
